ContextMenuManager

纯粹的Windows右键菜单管理程序

⏰ 最后更新： Aug 28, 2021 | ✅ 当前版本：3.3.3.1

🌟 核心功能

  • 启用或禁用右键菜单项目如：文件、文件夹、新建、发送到、打开方式、自定义文件格式、IE浏览器、WinX

  • 使用了系统提供的键值进行隐藏,不暴力删除备份

  • 兼容性能好

🎯 适用场景

  1. ❗️右键列表过长影响平时使用

  2. 希望为右键添加新功能

  3. 自定义爱好者 ♥️

🛠️ 配置方法

GitHub页面

  1. 找到相应版本的release文件下载

  2. fork到自己的仓库使用

官网

按照网站指引操作

😋 使用方法

打开程序，选择需要的右键功能即可

🔳 软件界面

Ditto

剪切板拯救者，开机自启常驻

⏰ 最后更新： 3 weeks ago | ✅ 当前版本：3.25.76.0

🌟 核心功能

  • 保存可以放置在复制板上的任何类型的信息，包括文本、图像、HTML、自定义格式

  • 使用了系统提供的键值进行隐藏,不暴力删除备份

  • 兼容性能好

🎯 适用场景

  1. 复制粘贴内容格式多

  2. 同一内容使用次数多

  3. 办公软件用户

🛠️ 配置方法

GitHub页面

  1. 找到相应版本的release文件下载

  2. fork到自己的仓库使用

官网

按照网站指引操作

😋 使用方法

  1. 运行 Ditto

  2. 将内容复制到剪贴板，例如在文本编辑器中选中文本后使用 Ctrl-C

  3. 通过点击系统托盘中的 Ditto 图标或按下快捷键（默认为 Ctrl + `）来打开 Ditto

  4. 双击或按 Enter 键选中项目，粘贴后即是选中内容

🔳 软件界面

Quicklook

空格查看文件，Windows cos macOS 😝

⏰ 最后更新： Jul 22 | ✅ 当前版本：4.1.1

🌟 核心功能

  • 允许用户只需按 Space 键，就能以很快的速度预览文件内容

🎯 适用场景

  1. 支持大量文件类型

  2. 使用极其流畅

  3. 严格遵循 GPL 许可证，永久免费

🛠️ 配置方法

GitHub页面

  1. 找到相应版本的release文件下载

  2. fork到自己的仓库使用

😋 使用方法

  1. 保持软件后台运行

  2. 鼠标选中欲预览文件

  3. Space 键预览文件信息

  4. ESC 键隐藏预览窗口

  5. Enter 键打开预览文件
